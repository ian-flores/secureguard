# Run a single guardrail

Run a single guardrail

## Usage

``` r
run_guardrail(guardrail, x)
```

## Arguments

- guardrail:

  A guardrail object.

- x:

  The input to check (string for input/code, any R object for output).

## Value

A
[`guardrail_result()`](https://ian-flores.github.io/secureguard/reference/guardrail_result.md).
